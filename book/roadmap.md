## 每章核心交付物

| Ch | 你构建的 | 你运行的 | 你理解的 |
|----|---------|---------|---------|
| 01 | agent_loop() | `python agent.py "list files"` | Agent 循环的本质 |
| 02 | TOOL_HANDLERS{} | 用 Agent 读写文件 | 工具分发模式 |
| 03 | Interactive REPL | 多轮对话 | 消息列表 = 记忆 |
| 04 | TodoManager | Agent 拆解复杂任务 | Plan-then-Execute |
| 05 | SkillLoader | Agent 按需加载知识 | 两层注入架构 |
| 06 | ContextCompressor | 超长任务不崩溃 | Token 经济学 |
| 07 | run_subagent() | 子 Agent 完成子任务 | 上下文隔离 |
| 08 | BackgroundManager | 异步执行慢操作 | 并发 vs 并行 |
| 09 | TaskGraph (DAG) | 持久化依赖任务 | 文件级状态管理 |
| 10 | TeamManager | 多 Agent 邮箱通信 | 持久化 vs 一次性 Agent |
| 11 | Protocols | 安全关闭和审批 | 请求-响应模式 |
| 12 | IDLE/WORK cycle | Agent 自主认领任务 | 自主的边界 |
| 13 | WorktreeManager | 多 Agent 隔离工作 | 控制面 vs 执行面 |
| 14 | SecurityLayer | 沙箱和确认机制 | 最小权限原则 |
| 15 | Observability | 追踪和调试 | 可观测性三支柱 |
| 16 | — (回顾) | 评估真实框架 | 架构决策能力 |
